# novel-outline

小说大纲规划专家技能 - 专为网文创作设计的大纲生成与规划系统。

## When to use

当需要创作小说大纲、规划整体剧情、设计人物关系、构建世界观时使用此技能。

适用场景：
- 从零开始创作全新小说
- 优化现有小说大纲
- 设计人物成长弧线
- 规划长篇连载剧情
- 设计世界观和力量体系

## Workflow

1. 理解创作需求：
   - 小说类型（玄幻/都市/仙侠/科幻等）
   - 目标平台（番茄/起点/晋江等）
   - 预期字数（短篇/中篇/长篇）
   - 更新频率（日更/周更等）

2. 分析目标市场：
   - 当前平台热门题材
   - 读者偏好分析
   - 竞争作品研究
   - 差异化定位

3. 设计核心要素：
   - 主角人设与金手指
   - 核心冲突与矛盾
   - 世界观与力量体系
   - 主要配角与反派

4. 规划剧情结构：
   - 开篇（黄金三章）
   - 中期发展（升级/打脸/情感）
   - 高潮转折
   - 结局设计

5. 制定章节规划：
   - 每章核心爽点
   - 伏笔埋设与回收
   - 节奏把控
   - 字数分配

## Rules

### 开篇设计（黄金三章）
- 第1章：300字内出核心冲突 + 金手指激活
- 第2章：第一次小爽点 + 世界观初步展示
- 第3章：第一个大冲突 + 读者期待感拉满

### 节奏把控
- 每300字一个小爽点
- 每1000字一个大爽点
- 绝不拖沓，情绪价值拉满
- 每章结尾必须留钩子

### 人设塑造
- 主角杀伐果断，不圣母
- 打脸要快，装逼要狠
- 人物成长弧线清晰
- 配角有血有肉，不是工具人

### 世界观构建
- 力量体系严谨，境界划分清晰
- 战力不崩，世界观自洽
- 升级体系有明确逻辑
- 背景设定服务于剧情

### 伏笔设计
- 每章至少埋设1-2个伏笔
- 伏笔要有层次感（短期/中期/长期）
- 伏笔回收要自然
- 长线剧情和短线爽点结合

## 平台适配

### 番茄小说
- 节奏极快，开篇即高潮
- 短句为主，口语化表达
- 爽点密集，情绪价值拉满
- 主角强势，打脸及时

### 起点中文网
- 世界观宏大，设定严谨
- 升级体系清晰，长线剧情
- 人物塑造立体，配角出彩
- 打斗场面精彩，技能有创意

### 晋江文学城
- 情感细腻，人物关系复杂
- 节奏适中，注重心理描写
- 人设鲜明，CP感强
- 文笔优美，氛围营造到位
