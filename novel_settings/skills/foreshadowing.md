# foreshadowing

伏笔设计与回收专家技能 - 专为网文创作设计的高级伏笔系统。

## When to use

当需要埋设伏笔、规划伏笔回收、确保剧情连贯性、设计剧情反转时使用此技能。

适用场景：
- 设计长篇连载的伏笔网络
- 确保伏笔自然不生硬
- 规划伏笔回收时机
- 设计剧情反转与伏笔呼应
- 检查伏笔是否遗漏

## Workflow

1. 伏笔规划阶段：
   - 分析整体剧情结构
   - 设计伏笔层次（短期/中期/长期）
   - 为每个伏笔设计埋设方式
   - 规划伏笔回收的最佳时机
   - 确保伏笔与主线剧情结合

2. 伏笔埋设阶段：
   - 选择合适的埋设位置
   - 设计自然的埋设方式
   - 确保伏笔不喧宾夺主
   - 为每个伏笔做标记记录
   - 控制伏笔的密集程度

3. 伏笔回收阶段：
   - 检查伏笔埋设记录
   - 选择最佳回收时机
   - 设计自然的回收方式
   - 确保回收有震撼力
   - 为回收后的剧情做铺垫

4. 剧情反转设计：
   - 利用伏笔设计反转
   - 确保反转有足够铺垫
   - 让反转既意外又合理
   - 利用多个伏笔交织
   - 设计多重反转层次

5. 伏笔检查与优化：
   - 检查是否有遗漏伏笔
   - 优化伏笔埋设方式
   - 调整伏笔回收时机
   - 移除冗余伏笔
   - 增强关键伏笔的效果

## Rules

### 伏笔层次设计

#### 短期伏笔（1-10章）
- 埋设目的：制造近期紧张感
- 埋设位置：章节开头、中间、结尾
- 回收时机：1-10章内
- 埋设方式：对话、细节、场景描写
- 例子：某人的异常举动、神秘物品、未说完的话

#### 中期伏笔（10-50章）
- 埋设目的：推动主线剧情发展
- 埋设位置：关键节点、重要事件
- 回收时机：10-50章内
- 埋设方式：人物背景、世界观设定、剧情暗示
- 例子：神秘组织的线索、主角的身世之谜、重要物品的来历

#### 长期伏笔（50章以上）
- 埋设目的：构建宏大世界观和主线
- 埋设位置：开篇、重要转折点
- 回收时机：50章以上甚至全书结尾
- 埋设方式：世界观设定、神话传说、预言、梦境
- 例子：世界的真相、终极boss的身份、主角的终极使命

### 伏笔埋设原则

#### 自然不生硬
- 伏笔要融入剧情，不能为了埋而埋
- 伏笔看起来要像正常的细节描写
- 读者第一次看不会注意到是伏笔
- 伏笔回收时读者会恍然大悟
- 避免用"注意了，这是伏笔"的方式

#### 有层次感
- 不同层次的伏笔要错开埋设
- 不要在同一时间密集埋设大量伏笔
- 伏笔之间要有联系，形成伏笔网络
- 重要伏笔要有多次暗示，加强记忆
- 次要伏笔点到为止，不要喧宾夺主

#### 有记录管理
- 每个伏笔都要有详细记录
- 记录包括：埋设位置、内容、类型、计划回收时间
- 定期检查伏笔记录，确保不遗漏
- 回收后及时更新记录状态
- 可以用工具辅助管理伏笔

### 伏笔回收原则

#### 时机选择
- 短期伏笔要及时回收，保持剧情紧张感
- 中期伏笔在关键节点回收，推动剧情发展
- 长期伏笔在高潮或结尾回收，产生震撼效果
- 回收时机要和剧情节奏配合
- 不要让伏笔埋太久忘记回收

#### 回收方式
- 回收要自然，不能生硬
- 最好通过剧情发展自然回收
- 回收时可以和新伏笔结合
- 重要伏笔的回收要有仪式感
- 可以设计多重回收，逐步揭示真相

#### 回收效果
- 回收要让读者有"原来是这样"的感觉
- 重要伏笔的回收要有震撼力
- 回收要推动剧情发展
- 回收可以为新剧情做铺垫
- 利用伏笔设计剧情反转

### 剧情反转设计

#### 利用伏笔
- 反转必须有足够的伏笔铺垫
- 反转不能凭空产生
- 利用多个伏笔交织制造反转
- 反转既要意外又要合理
- 读者回顾时能发现所有线索

#### 多层反转
- 可以设计多重反转，层层递进
- 每次反转都要有伏笔支撑
- 反转层次要清晰，不要混乱
- 最终反转要有震撼力
- 反转后要给读者消化时间

#### 反转效果
- 反转要改变读者对剧情的认知
- 反转要影响后续剧情发展
- 反转要让人物关系更复杂
- 反转要揭示更深层的真相
- 反转要让故事更有深度

### 伏笔网络构建

#### 伏笔之间的联系
- 伏笔之间要有逻辑联系
- 形成伏笔网络，互相支撑
- 一个伏笔的回收可以触发另一个伏笔
- 利用伏笔构建复杂剧情
- 伏笔网络要服务于主线

#### 伏笔与主线的关系
- 所有伏笔最终都要服务于主线
- 不要设计与主线无关的伏笔
- 伏笔要推动主线剧情发展
- 主线剧情要利用伏笔
- 伏笔和主线要交织在一起

#### 伏笔的密度控制
- 不要在同一时间埋设太多伏笔
- 伏笔密度要和剧情节奏配合
- 紧张情节时伏笔要少
- 舒缓情节时可以多埋一些
- 重要章节可以集中埋设关键伏笔

### 伏笔检查与优化

#### 遗漏检查
- 定期检查伏笔记录
- 确保每个伏笔都有回收计划
- 避免伏笔埋太久忘记
- 回收后及时更新状态
- 可以用工具辅助检查

#### 冗余清理
- 移除对剧情没用的伏笔
- 合并重复的伏笔
- 简化过于复杂的伏笔
- 移除喧宾夺主的伏笔
- 保留核心重要的伏笔

#### 效果优化
- 增强关键伏笔的埋设效果
- 优化伏笔回收时机
- 让伏笔更自然
- 让回收更震撼
- 提升整体伏笔质量
