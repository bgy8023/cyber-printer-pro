## 4. 核心技术亮点

### 4.1 DeerFlow 2.0 架构
```
主脑（调度层）
    ↓ 只做任务拆分，不执行
Agent 池（执行层）
    ↓ 独立执行，互不干扰
结果聚合（输出层）
```

**优势：**
- 职责清晰，易于维护
- 支持动态扩展 Agent
- 容错性强，单个 Agent 失败不影响整体

### 4.2 双层记忆宫殿
```
固定记忆（不可变）
    ├── 人物档案
    ├── 世界观设定
    └── 全书大纲

动态记忆（自动更新）
    ├── 已生成章节
    ├── 已埋设伏笔
    └── 剧情节点
```

**优势：**
- 内存占用降低 70%+
- 支持无状态启动
- 数据持久化安全可靠

### 4.3 中转池支持
- 支持配置多个 API 端点
- 自动故障切换
- 负载均衡分配请求

### 4.4 动态并行调度
- 自动计算最优线程数
- 任务优先级支持
- 依赖关系管理
